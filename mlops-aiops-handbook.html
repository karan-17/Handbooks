MLOps Fundamentals

Definition and Core Concepts

MLOps is a set of practices that automates and simplifies machine learning workflows and deployments, aiming to deploy and maintain machine learning models in production reliably and efficiently. It leverages three contributing disciplines: machine learning, software engineering (especially DevOps), and data engineering.

Core MLOps Principles for 2025

  • Automation and CI/CD: Systematic automation of the ML lifecycle and model deployment
  • Versioning and Reproducibility: Track changes in data, models, and code for reproducible results
  • Continuous Monitoring: Observability systems to track metrics and detect drift
  • Collaboration: Cross-functional teams including data scientists, ML engineers, and DevOps

MLOps Maturity Levels

Level 0: Manual Process

Manual processes, no automation, data science driven approach with limited collaboration.

Level 1: ML Pipeline Automation

Automated ML pipelines, continuous training, and experimental-operational symmetry.

Level 2: CI/CD Pipeline Automation

Full CI/CD automation with continuous training, delivery, and deployment capabilities.

# Example drift detection implementation
import pandas as pd
from scipy import stats
from typing import Dict, Any

class DriftDetector:
    def __init__(self, reference_data: pd.DataFrame):
        self.reference_stats = self._calculate_stats(reference_data)
    
    def _calculate_stats(self, data: pd.DataFrame) -> Dict[str, Any]:
        return {
            'mean': data.mean(),
            'std': data.std(),
            'quantiles': data.quantile([0.25, 0.5, 0.75])
        }
    
    def detect_drift(self, new_data: pd.DataFrame, threshold: float = 0.05) -> Dict[str, bool]:
        drift_results = {}
        
        for column in self.reference_stats['mean'].index:
            # Kolmogorov-Smirnov test for distribution comparison
            ks_stat, p_value = stats.ks_2samp(
                self.reference_data[column], 
                new_data[column]
            )
            drift_results[column] = p_value < threshold
            
        return drift_results

# Example anomaly detection with isolation forest
from sklearn.ensemble import IsolationForest
import pandas as pd
import numpy as np

class AnomalyDetector:
    def __init__(self, contamination=0.1):
        self.model = IsolationForest(contamination=contamination, random_state=42)
        self.fitted = False
    
    def fit(self, normal_data):
        """Train the model on normal operational data"""
        self.model.fit(normal_data)
        self.fitted = True
    
    def predict_anomalies(self, new_data):
        """Predict anomalies in new data"""
        if not self.fitted:
            raise ValueError("Model must be fitted first")
        
        predictions = self.model.predict(new_data)
        anomaly_scores = self.model.decision_function(new_data)
        
        return {
            'is_anomaly': predictions == -1,
            'anomaly_score': anomaly_scores
        }
