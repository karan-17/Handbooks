🎯 Overview & Introduction

Understanding RAG evaluation and why it's crucial for building reliable AI systems

🤖 What is RAG (Retrieval-Augmented Generation)?

Retrieval-Augmented Generation (RAG) is an AI technique that combines the power of large language models with external knowledge retrieval. It addresses the limitations of LLMs by:

  • Accessing up-to-date information beyond training data
  • Reducing hallucinations through grounded responses
  • Providing domain-specific knowledge
  • Enabling citations and source attribution
📏 The RAG Evaluation Challenge

Evaluating RAG systems is complex because you need to assess multiple components:

📚 Retrieval Quality

How well does the system find relevant documents?

  • Context Relevance
  • Context Precision
  • Context Recall
⚙️ Generation Quality

How well does the model generate responses?

  • Answer Relevance
  • Faithfulness
  • Answer Correctness

❓ Why Evaluate RAG Systems?

Understanding the critical importance of systematic RAG evaluation

🎯 Quality Assurance

Ensure your RAG system provides accurate, relevant, and reliable responses.

Key Insight: Without proper evaluation, you can't know if your RAG system is actually helping or potentially providing misleading information.
🔧 System Optimization

Identify bottlenecks and optimize different components of your RAG pipeline.

  • Embedding model selection
  • Chunking strategy optimization
  • Retrieval parameter tuning
  • Prompt engineering
📊 Performance Monitoring

Track system performance over time and detect degradation.

Production Reality: RAG systems can degrade over time due to data drift, model updates, or changing user patterns.
🏆 Competitive Analysis

Compare different approaches and architectures objectively.

  • A/B testing different models
  • Architecture comparison
  • Cost-performance trade-offs

📊 RAGAS Framework

Retrieval Augmented Generation Assessment - A comprehensive evaluation framework for RAG systems

R

RAGAS

Version 0.1+ | Hugging Face

RAGAS is a framework that provides reference-free evaluation metrics for RAG systems. It evaluates both retrieval and generation components without requiring ground truth answers.

Key Advantage: RAGAS can evaluate your RAG system using only questions, contexts, and generated answers - no need for human-annotated ground truth!
🎯 RAGAS Core Metrics
📚 Context Precision 0.85

Measures how relevant the retrieved contexts are to the question.

Context Precision = (Relevant contexts in top-k) / k
🔄 Context Recall 0.78

Measures how well the retrieval system captures all relevant information.

Context Recall = (Retrieved relevant contexts) / (All relevant contexts)
✨ Faithfulness 0.92

Measures how factually accurate the answer is relative to the context.

Faithfulness = (Faithful statements) / (Total statements)
🎯 Answer Relevancy 0.81

Measures how relevant the answer is to the question.

Answer Relevancy = cosine_similarity(question_embedding, answer_embedding)
📝 Answer Semantic Similarity 0.73

Measures semantic similarity between generated and ground truth answers.

Semantic Similarity = cosine_similarity(answer_embedding, ground_truth_embedding)
✅ Answer Correctness 0.79

Combines semantic similarity with factual correctness.

Answer Correctness = w1 × Semantic_Similarity + w2 × Factual_Similarity

📊 Batch Evaluation with Progress Tracking

from tqdm import tqdm
import pandas as pd

def batch_evaluate_rag(data_file, batch_size=10):
    """Evaluate RAG system in batches with progress tracking"""
    
    # Load data
    df = pd.read_csv(data_file)
    total_batches = len(df) // batch_size + 1
    results = []
    
    for i in tqdm(range(0, len(df), batch_size), 
                     desc="Evaluating RAG"):
        
        # Get batch
        batch = df[i:i+batch_size]
        
        # Create dataset for this batch
        batch_dataset = Dataset.from_dict({
            "question": batch["question"].tolist(),
            "contexts": batch["contexts"].apply(lambda x: [x]).tolist(),
            "answer": batch["answer"].tolist(),
            "ground_truth": batch["ground_truth"].tolist()
        })
        
        # Evaluate batch
        batch_result = evaluate(
            dataset=batch_dataset,
            metrics=[faithfulness, answer_relevancy, context_precision, answer_correctness]
        )
        
        results.append(batch_result)
    
    # Aggregate results
    final_results = {
        metric: sum(r[metric] for r in results) / len(results)
        for metric in results[0].keys()
    }
    
    return final_results

📈 Continuous Monitoring Setup

import schedule
import time
from datetime import datetime

class RAGMonitor:
    """Continuous monitoring for RAG systems"""
    
    def __init__(self, rag_system, evaluation_dataset):
        self.rag_system = rag_system
        self.evaluation_dataset = evaluation_dataset
        self.history = []
    
    def evaluate_system(self):
        """Run evaluation and store results"""
        timestamp = datetime.now()
        
        # Generate fresh answers
        fresh_answers = []
        for question in self.evaluation_dataset["question"]:
            answer = self.rag_system.generate_answer(question)
            fresh_answers.append(answer)
        
        # Create evaluation dataset
        eval_data = Dataset.from_dict({
            "question": self.evaluation_dataset["question"],
            "contexts": self.evaluation_dataset["contexts"],
            "answer": fresh_answers,
            "ground_truth": self.evaluation_dataset["ground_truth"]
        })
        
        # Evaluate
        results = evaluate(
            dataset=eval_data,
            metrics=[faithfulness, answer_relevancy, context_precision]
        )
        
        # Store results with timestamp
        results["timestamp"] = timestamp
        self.history.append(results)
        
        # Check for degradation
        self.check_performance_degradation(results)
        
        return results
    
    def check_performance_degradation(self, current_results):
        """Alert if performance degrades significantly"""
        if len(self.history) < 2:
            return
        
        previous = self.history[-2]
        threshold = 0.1  # 10% degradation threshold
        
        for metric in ["faithfulness", "answer_relevancy", "context_precision"]:
            degradation = previous[metric] - current_results[metric]
            if degradation > threshold:
                print(f"⚠️ ALERT: {metric} degraded by {degradation:.3f}")

# Setup monitoring
monitor = RAGMonitor(your_rag_system, evaluation_dataset)
schedule.every().hour.do(monitor.evaluate_system)

# Run monitoring
while True:
    schedule.run_pending()
    time.sleep(60)

from deepeval.metrics import BaseMetric
from deepeval.test_case import LLMTestCase
from typing import Optional

class CustomRAGMetric(BaseMetric):
    """Custom metric for domain-specific RAG evaluation"""
    
    def __init__(
        self,
        threshold: float = 0.5,
        model: Optional[str] = None,
        include_reason: bool = True
    ):
        self.threshold = threshold
        self.model = model
        self.include_reason = include_reason
    
    def measure(self, test_case: LLMTestCase) -> float:
        """Implement custom measurement logic"""
        
        # Access test case components
        question = test_case.input
        answer = test_case.actual_output
        context = test_case.retrieval_context
        
        # Implement your custom scoring logic
        score = self._calculate_domain_score(question, answer, context)
        
        # Set score and reason
        self.score = score
        self.reason = f"Domain-specific score: {score:.3f}"
        self.success = score >= self.threshold
        
        return score
    
    def _calculate_domain_score(self, question, answer, context):
        """Custom domain-specific scoring logic"""
        # Example: Check for domain-specific keywords
        domain_keywords = ["technical", "specification", "implementation"]
        
        keyword_score = sum(
            1 for keyword in domain_keywords 
            if keyword in answer.lower()
        ) / len(domain_keywords)
        
        # Add context relevance check
        context_score = len(
            set(answer.lower().split()) & 
            set(" ".join(context).lower().split())
        ) / len(answer.split())
        
        return (keyword_score + context_score) / 2
    
    def is_successful(self) -> bool:
        return self.success
    
    @property
    def __name__(self):
        return "Domain Relevance"

📊 Performance Regression Testing

import json
from pathlib import Path
from deepeval.dataset import EvaluationDataset

class RAGRegressionTester:
    """Test for performance regression in RAG systems"""
    
    def __init__(self, baseline_file: str = "baseline_scores.json"):
        self.baseline_file = Path(baseline_file)
        self.baseline_scores = self._load_baseline()
    
    def _load_baseline(self):
        """Load baseline performance scores"""
        if self.baseline_file.exists():
            with open(self.baseline_file) as f:
                return json.load(f)
        return {}
    
    def save_baseline(self, scores: dict):
        """Save current scores as baseline"""
        with open(self.baseline_file, 'w') as f:
            json.dump(scores, f, indent=2)
    
    def test_regression(self, current_scores: dict, threshold: float = 0.05):
        """Test for performance regression"""
        regressions = []
        
        for metric, current_score in current_scores.items():
            if metric in self.baseline_scores:
                baseline_score = self.baseline_scores[metric]
                degradation = baseline_score - current_score
                
                if degradation > threshold:
                    regressions.append({
                        "metric": metric,
                        "baseline": baseline_score,
                        "current": current_score,
                        "degradation": degradation
                    })
        
        return regressions
    
    def run_regression_test(self, dataset: EvaluationDataset):
        """Run full regression test"""
        # Run current evaluation
        current_results = evaluate(
            test_cases=dataset.to_test_cases(),
            metrics=[
                FaithfulnessMetric(),
                AnswerRelevancyMetric(),
                ContextualPrecisionMetric()
            ]
        )
        
        # Check for regressions
        regressions = self.test_regression(current_results)
        
        if regressions:
            error_msg = "Performance regression detected:\n"
            for reg in regressions:
                error_msg += f"- {reg['metric']}: {reg['baseline']:.3f} → {reg['current']:.3f} (Δ{reg['degradation']:.3f})\n"
            
            raise AssertionError(error_msg)
        
        return current_results

from trulens_eval.feedback import Feedback
from trulens_eval.feedback.provider.openai import OpenAI
import re

openai = OpenAI()

# Custom feedback for technical accuracy
def technical_accuracy(question: str, answer: str, context: str) -> float:
    """Custom function to assess technical accuracy"""
    
    prompt = f"""
    Assess the technical accuracy of this answer given the context.
    
    Question: {question}
    Context: {context}
    Answer: {answer}
    
    Rate the technical accuracy on a scale of 0.0 to 1.0 where:
    - 1.0 = Completely technically accurate
    - 0.8 = Mostly accurate with minor issues  
    - 0.6 = Somewhat accurate but has some errors
    - 0.4 = Limited accuracy, several errors
    - 0.2 = Poor accuracy, many errors
    - 0.0 = Completely inaccurate
    
    Provide only the numeric score.
    """
    
    response = openai.client.chat.completions.create(
        model="gpt-3.5-turbo",
        messages=[{"role": "user", "content": prompt}],
        temperature=0.0
    )
    
    # Extract numeric score
    score_text = response.choices[0].message.content.strip()
    try:
        score = float(re.findall(r'[0-9]*\.?[0-9]+', score_text)[0])
        return min(max(score, 0.0), 1.0)  # Clamp to [0,1]
    except:
        return 0.0

# Create custom feedback
technical_feedback = Feedback(
    technical_accuracy,
    name="Technical Accuracy"
).on_input().on_output().on(
    TruLlama.select_source_nodes().node.text.collect()
)

# Length-based feedback
answer_length = Feedback(
    lambda output: min(len(output.split()) / 50, 1.0),  # Normalize to ~50 words
    name="Answer Length"
).on_output()

# Combine with standard feedbacks
all_feedbacks = feedbacks + [technical_feedback, answer_length]

🔄 Continuous Monitoring Setup

import schedule
import time
from datetime import datetime
import logging

class TruLensMonitor:
    """Continuous monitoring with TruLens"""
    
    def __init__(self, app_name, alert_thresholds):
        self.app_name = app_name
        self.alert_thresholds = alert_thresholds
        self.tru = Tru()
        
    def check_performance(self):
        """Check recent performance and send alerts if needed"""
        
        # Get recent records (last hour)
        records = self.tru.get_records_and_feedback(
            app_ids=[self.app_name],
            limit=100  # Adjust based on traffic
        )
        
        if not records:
            return
            
        df = pd.DataFrame(records)
        
        # Calculate recent averages
        recent_scores = {
            "groundedness": df["Groundedness"].mean(),
            "answer_relevance": df["Answer Relevance"].mean(),
            "context_relevance": df["Context Relevance"].mean()
        }
        
        # Check for alerts
        for metric, score in recent_scores.items():
            if score < self.alert_thresholds.get(metric, 0.7):
                self._send_alert(metric, score)
    
    def _send_alert(self, metric, score):
        """Send performance alert"""
        message = f"⚠️ ALERT: {metric} dropped to {score:.3f} for app {self.app_name}"
        logging.warning(message)
        # Add your notification logic here (email, Slack, etc.)
    
    def start_monitoring(self):
        """Start continuous monitoring"""
        schedule.every(15).minutes.do(self.check_performance)
        
        while True:
            schedule.run_pending()
            time.sleep(60)

# Setup monitoring

    app_name="RAG_v1",
    alert_thresholds={

📊 Advanced Analytics

import matplotlib.pyplot as plt
import seaborn as sns

def analyze_performance_trends(app_name, days=7):
    """Analyze performance trends over time"""
    
    # Get records
    tru = Tru()
    records = tru.get_records_and_feedback(app_ids=[app_name])
    
    df = pd.DataFrame(records)
    df["timestamp"] = pd.to_datetime(df["timestamp"])
    
    # Filter to last N days
    cutoff = datetime.now() - pd.Timedelta(days=days)
    recent_df = df[df["timestamp"] >= cutoff]
    
    # Resample to hourly averages
    hourly_scores = recent_df.set_index("timestamp").resample("H").agg({
        "Groundedness": "mean",
        "Answer Relevance": "mean",
        "Context Relevance": "mean"
    })
    
    # Plot trends
    plt.figure(figsize=(12, 8))
    
    for i, metric in enumerate(hourly_scores.columns):
        plt.subplot(2, 2, i+1)
        plt.plot(hourly_scores.index, hourly_scores[metric])
        plt.title(f"{metric} Over Time")
        plt.ylabel("Score")
        plt.xticks(rotation=45)
    
    plt.tight_layout()
    plt.show()
    
    # Performance summary
    print(f"Performance Summary for {app_name} (Last {days} days):")
    print(hourly_scores.describe())
    
    return hourly_scores

# Run analysis
trends = analyze_performance_trends("RAG_v1", days=7)
