Software Technology (ST) - Complete Syllabus Handbook

🎓 Comprehensive AI & Software Technology Curriculum

Master AI fundamentals, agentic systems, cloud computing, and modern software development practices. From machine learning basics to advanced AI agent architectures and production deployment.

Course Overview

This comprehensive Software Technology course covers the complete spectrum of modern AI and software development, from foundational concepts to advanced implementation and deployment. The curriculum is designed to provide both theoretical understanding and practical skills for building production-ready AI systems.

Learning Objectives

  • AI Foundations: Understand machine learning fundamentals and neural network architectures
  • LLM Mastery: Deep knowledge of transformer models, attention mechanisms, and modern LLMs
  • Agentic AI: Design and implement intelligent agents and multi-agent systems
  • System Design: Architect scalable AI systems for production environments
  • Cloud & DevOps: Deploy and manage AI applications using modern cloud platforms
  • Production Skills: MLOps, monitoring, optimization, and best practices
